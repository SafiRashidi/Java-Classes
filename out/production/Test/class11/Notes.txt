2D Arrays:
    Are arrays that contain 1D array.
Why we should learn about 2D array?
    Because they allow us to store multiple 1D arrays which give us more flexibility to store data.
    for exp: if we have to store data from Excel file we can use 2D Arrays.
Syntax to create 2D arrays:
First Way:
    DataType [] [] nameOfArray= new DataType[size of Rows] [];
Second Way:
    DataType [] [] nameOfArray= {{data,data,data,
